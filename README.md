## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript based backend application for Dragondelve Tempest system for using The Brute Force GURPS extension Setting

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
